Secure contact form for WordPress. Uses end-to-end encryption to send user information. Not even your hosting provider can view the content. 

== Description ==

Let users send you information in a secure way. Uses I.CX messaging service to encrypt user content in their own web browsers before sending to you.

== Installation ==

Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page. Then go to Settings and select “Encrypted Contact Form” to create a new page.

== Frequently Asked Questions ==

Are there any additional requirements? 

All messages are sent to your I.CX username. They are encrypted with your public I.CX key. I.CX accounts are free and easy to sign up for. They require no personal information.
